A simple Python game built with the Turtle module where the player has to guess the country based on the displayed flag.

The game randomly draws a European flag on the screen and the player must type the correct country name.
Each correct answer gives one point, while wrong answers cost a life.

At the end, the player receives a grade based on their score.

Features

Random flag selection

Score and lives system

Multiple European flags drawn with Turtle graphics

Simple and fun learning game

How to play

Run the script

Watch the flag appear on the screen

Type the country name in the console

Try to get the highest score!
